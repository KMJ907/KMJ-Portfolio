# Codexora Portfolio

> HTML / CSS / JavaScript로 제작한 개인 포트폴리오 웹사이트

![Portfolio Preview](assets/images/profile.png)

---

## 📌 About

안녕하세요.

**Codexora Portfolio**는 저를 소개하기 위해 직접 제작한 개인 포트폴리오 사이트입니다.

React, Next.js와 같은 프레임워크 없이  
순수 **HTML, CSS, JavaScript**를 활용하여 제작하였으며,

웹 개발 과정에서 배운 기술과 진행했던 프로젝트, 활동 내용을 한 곳에서 확인할 수 있도록 구성했습니다.

---

## ✨ Features

- 🌙 Dark Theme 기반 디자인
- 📱 반응형 웹 디자인
- 🎨 Glassmorphism UI
- 🚀 부드러운 Scroll Animation
- 📊 Scroll Progress Bar
- 📌 Active Navigation
- 📱 Mobile Navigation
- ⬆ Back To Top 버튼
- 🔗 프로젝트 및 활동 소개
